# Listingify 当前用户操作手册

适用日期：2026-04-30

本文面向当前 Web 工作台已实现功能，重点说明 SHEIN 全托管发品工作流如何操作。SHEIN 单款真实发布链路已跑通，真实发布前仍需要确认 SHEIN 平台对接配置、商品字段、图片、尺码、价格、包装和发布校验结果。

## 1. 登录系统

1. 启动后端和前端：

```bash
npm run web:server
npm run web:dev
```

2. 打开 Vite 地址，例如 `http://localhost:5173`。
3. 使用管理员账号登录。
4. 系统不再内置默认管理员账号。首次部署请通过 `npm run admin:create -- --username <账号> --password <强密码>` 创建管理员，或同时配置 `LISTINGIFY_ADMIN_USERNAME` 和 `LISTINGIFY_ADMIN_PASSWORD` 后启动服务自动初始化。

## 2. 工作流总览

SHEIN 发品的当前主流程是：

```text
数据中心
  -> 商品档案确认 MDM/深绘源头数据
  -> SHEIN 商品分桶做平台字段清洗
  -> SHEIN 发布草稿箱生成和维护草稿
  -> 上新批次组织多草稿
  -> 批量校验和批量发布
  -> 发布任务查看结果、失败原因和状态同步
```

各页面职责：

- 「商品档案」：查看源头商品数据，不直接承载 SHEIN 业务字段。
- 「SHEIN 商品分桶」：维护 SHEIN 平台业务字段，创建发布草稿。
- 「SHEIN 发布草稿箱」：维护草稿版本、校验字段、图片、尺码、价格、包装并提交发布。
- 「上新批次」：把多个草稿归入同一批次，便于批量推进。
- 「发布任务」：追踪真实提交、失败原因、重试和状态同步。

## 3. 数据中心

### 商品档案

入口：数据中心 -> 商品档案

用途：

- 查看从 MDM 和深绘同步后的商品聚合数据。
- 查看 SPU、SKC、SKU、品牌、季节、类目、颜色、尺码、图片、深绘内容包等源头信息。
- 支持分页、每页数量设置、批量款号搜索和多字段筛选。

操作建议：

1. 用款号搜索目标商品，例如 `201122104105`。
2. 进入详情页确认 MDM SKU、SKC 款色、尺码、深绘图片和内容包是否完整。
3. 源头数据不在此处改写，SHEIN 平台字段进入「SHEIN 商品分桶」维护。

### MDM 商品主数据

入口：数据中心 -> MDM 商品主数据

用途：

- 查看 MDM 原始 SPU/SKC/SKU 主数据。
- 用于核对商家 SKU、颜色、尺码、品牌、类目、年份季节等源头字段。

### 深绘内容包

入口：数据中心 -> 深绘内容包

用途：

- 查看深绘同步的标题、图片、详情页、尺码表和字段池。
- 支撑草稿阶段的 AI 字段推荐、图片素材选择和尺码表映射。

### 图片素材库

入口：数据中心 -> 图片素材库

用途：

- 查看 MDM/深绘/TMALL/VIP 等来源图片。
- 支持按款号、款色、图片类型、来源平台、尺寸等信息辅助查图。
- 草稿详情页的素材弹窗会复用这里的相关图片。

## 4. 规则中心

### 类目映射

入口：规则中心 -> 类目映射

用途：

- 维护 `MDM 中类 + 小类 + 性别 + 年龄段 -> SHEIN 末级类目` 的组合规则。
- 支持 AI 推荐、人工审核和规则落库。

建议：

- 当同一款不同款色可能因模特性别、颜色或使用场景进入不同 SHEIN 类目时，在 AI 审核时按 SKC 图片和款色上下文判断。
- 对高频类目优先沉淀确定性规则，AI 用于缺规则或边界场景推荐。

### SHEIN 尺码转换

入口：规则中心 -> SHEIN 尺码转换

用途：

- 维护内部尺码到 SHEIN 发布尺码的映射。
- 支持表格导入、导出、批量搜索、新增、编辑和删除。

草稿中的发布尺码以转换后的 SHEIN 尺码为准，并与勾选发布的 SKU 保持一致。

### 包装规则

入口：规则中心 -> 包装规则

用途：

- 维护包装长、宽、高、默认包装策略等规则。
- 草稿 SKU 行可以读取规则，也支持人工批量填充和修改。

### 价格规则

入口：规则中心 -> 价格规则

用途：

- 维护供货价、折扣、倍率、币种和价格相关规则。
- 草稿 SKU 行可以维护供货价，发布前需要人工确认。

### SHEIN 低倍率清单

入口：规则中心 -> SHEIN 低倍率清单

用途：

- 维护 SHEIN 低倍率商品清单。
- 支持表格导入、导出、批量搜索和增删改查。

## 5. SHEIN 商品分桶

入口：上新工作 -> SHEIN 商品分桶

用途：

- 将商品从源头档案挑选进 SHEIN 平台业务池。
- 维护 SHEIN 类目、英文标题、价格、包装、字段完整度、图片状态和发布历史。
- 从一个 SHEIN 商品派生多个发布草稿。

常用操作：

1. 在搜索框输入款号、标题、品牌或类目信息。
2. 使用品牌、类目、分桶状态、类目状态、完整度状态、图片状态等筛选。
3. 勾选目标商品。
4. 点击 AI 推荐，系统会结合商品档案、深绘字段和 SHEIN 类目/枚举上下文推荐类目或字段值。
5. 确认平台类目和关键字段。
6. 点击创建草稿，进入 SHEIN 发布草稿箱或单款草稿详情。

说明：

- 商品档案保留 MDM/深绘源头数据。
- SHEIN 相关业务字段放在商品分桶和发布草稿中，便于后续扩展其他平台。
- 同一个 SHEIN 商品可以派生多个草稿版本，每个草稿独立流转状态。

## 6. SHEIN 发布草稿箱

入口：上新工作 -> SHEIN 发布草稿箱

用途：

- 查看和维护所有 SHEIN 发布草稿。
- 批量校验、批量发布、处理阻断项。
- 进入单款草稿详情做完整编辑。

列表能力：

- 显示商品图片、SPU、标题、品牌、类目、版本、状态、字段完整度、阻断项、SKC/SKU 数量和更新时间。
- 支持平台选择，目前主要为 SHEIN。
- 支持按款号批量搜索。
- 支持按 SHEIN 类目筛选。
- 支持分页和每页数量设置。

草稿状态：

- 草稿：仍在编辑。
- 待补齐：存在字段、图片、价格、尺码或包装缺口。
- 待校验：准备进行发布前校验。
- 校验失败：有阻断项。
- 可发布：通过发布前校验。
- 发布中：已提交发布任务。
- 已提交：平台已接收。
- 发布失败：接口或平台返回失败。
- 已暂停/已归档：人工暂停或归档的草稿。

批量发布：

1. 勾选一个或多个草稿。
2. 点击批量校验。
3. 如存在阻断项，弹窗会展示需要快速调整的字段。
4. 在弹窗中补齐字段、SKU 毛重或图片确认。
5. 再次校验通过后，点击批量发布。
6. 发布后进入「发布任务」查看提交结果。

## 7. 单款发布草稿详情

入口：SHEIN 发布草稿箱 -> 点击草稿

页面按 `SPU -> SKC -> SKU` 维度组织。

### SPU 维度

可维护内容：

- SHEIN 类目。
- 中文标题、英文标题。
- 商品属性和类目字段。
- AI 推荐字段。
- 草稿整体保存。

操作：

- 点击 SHEIN 类目选择，打开树形多级类目弹窗。
- 选择叶子类目后，页面会按类目展示字段、枚举、图片规则和尺码模板。
- 英文标题可由中文标题 AI 翻译，也可人工修改。
- 对枚举较多的字段，使用下拉或选择控件确认，不用平铺大量单选项。

### SKC 维度

可维护内容：

- 是否发布该款色。
- 款色图。
- 颜色属性。
- 图片规则完成情况。
- SKC 下可发布 SKU。

说明：

- 同一个 SPU 下不同 SKC 可以因为款色图中的模特性别、颜色或业务判断进入不同类目或不同发布策略。
- 当前草稿页按 SKC 展示款色图，便于人工识别。

### SKU 维度

可维护内容：

- MDM SKU。
- 内部尺码。
- SHEIN 发布尺码。
- 是否发布该 SKU。
- 供货价。
- SKU 毛重。
- 包装长、宽、高。

批量维护：

- 在「按 SKC 选择发布尺码」模块中，可对供货价、SKU 毛重、包装长宽高进行批量填充。
- 毛重当前按 SKU 维度维护，示例联调中可先填 `500g`。
- 发布尺码以 SHEIN 转换尺码为准，并且必须与勾选发布 SKU 保持一致。

### 尺码表

可维护内容：

- SHEIN 类目所需尺码模板。
- 从深绘尺码表映射到 SHEIN 模板字段后的结果。

说明：

- 草稿箱不直接平铺深绘全部尺码表。
- 应以 SHEIN 类目要求的尺码字段为目标，把深绘尺码数据映射进去。

### 图片模块

可维护内容：

- SPU/SKC/SKU 维度所需图片。
- 图片类型、来源、尺寸、文件大小和合规状态。
- 本地上传图片。
- 从图片素材库选择图片。
- 图片确认、编辑和删除。

素材库选择：

1. 点击从素材库选择。
2. 弹窗展示与当前款号和款色相关的图片。
3. 默认勾选来源平台 TMALL 和 VIP。
4. 默认不选择「商详模块」「商详截图」。
5. 可按图片类型和来源平台筛选。
6. 图片卡片会显示来源平台、类型、尺寸和文件大小。
7. 合规图片优先展示。
8. 可一键剔除不符合 SHEIN 图片尺寸或大小要求的图片。
9. 选择后保存到草稿图片模块。

## 8. 上新批次

入口：上新工作 -> 上新批次

用途：

- 把多个发布草稿组织到一个 SHEIN 上新批次中。
- 以批次维度查看草稿数、SKC/SKU 范围、阻断数量、发布中数量、审核中数量、通过数量和失败数量。

常用操作：

1. 新建批次。
2. 选择需要归入批次的草稿。
3. 进入批次详情查看草稿列表和发布进度。
4. 对批次内草稿进行批量校验和批量发布。

关系说明：

- SHEIN 商品分桶：平台业务商品池。
- 发布草稿：从商品分桶派生出的具体发布版本。
- 上新批次：多个草稿的批量管理容器。
- 发布任务：草稿提交平台后产生的执行记录。

## 9. 发布任务

入口：上新工作 -> 发布任务

用途：

- 查看真实发布接口提交记录。
- 查看请求 payload、响应、traceId、失败原因和关联草稿版本。
- 对失败任务进行重试或后续状态同步。

常用操作：

1. 按任务状态、款号或批次搜索。
2. 点击任务进入详情。
3. 查看 SHEIN 返回的错误信息。
4. 返回草稿详情修正字段、图片、尺码、价格或包装。
5. 重新提交发布。

## 10. 系统管理

### 平台对接

入口：系统管理 -> 平台对接

用途：

- 配置 SHEIN 店铺或应用对接信息。
- 管理 base URL、环境、业务模式、openKeyId、secretKey、appId、appSecretKey、默认账号和状态。
- 密钥在接口响应中脱敏展示。

建议：

- 生产密钥不要写入代码或文档。
- 本地 `.env` 和平台对接配置二选一或按优先级使用。
- 配置变更后通过测试状态确认可用性。

### 用户管理

入口：系统管理 -> 用户管理

用途：

- 创建用户。
- 分配管理员、运营、只读等角色。
- 启停账号。
- 重置密码。

### 同步任务

入口：系统管理 -> 同步任务

用途：

- 查看 MDM、深绘、SHEIN 元数据和发布相关任务记录。
- 追踪同步时间、状态和结果。

### 操作日志

入口：系统管理 -> 操作日志

用途：

- 查看登录、发布、用户管理、平台配置等关键操作。
- 支持按模块、动作和关键词筛选。

## 11. 当前注意事项

- SHEIN 单款真实发布已跑通；遇到接口错误或审核驳回时，先查看发布任务详情，再回到草稿详情修正字段并生成新版本重提。
- `fill_in_standard_list.show=false` 的 SHEIN 字段不要提交。
- `attribute_status=3` 的属性必须补齐。
- 图片规则和属性模板按 SHEIN 类目变化，切换类目后需要重新检查字段和图片。
- SKU 毛重以 SKU 维度维护，不再按 SPU 平均值。
- 发布前必须确认发布颜色和尺码，未勾选的 SKC/SKU 不应进入发布 payload。
- 商品档案只保留源头数据，平台字段不要回写到商品档案。
