# SHEIN 全托管上新中台 MVP PRD

## 1. 文档信息

| 项目 | 内容 |
| --- | --- |
| 产品方向 | 商品中台 + 平台刊登中台 + ERP 能力演进 |
| MVP 范围 | SHEIN 全托管上新 |
| 未来预留 | TEMU 等多平台适配 |
| 数据来源 | MDM SPU/SKU 接口、深绘同步接口、人工补齐数据 |
| 文档日期 | 2026-04-24 |

## 2. 背景

当前跨境电商上新流程依赖 Excel 汇总：MDM 提供商品主数据，深绘提供标题、详情素材、尺码表和部分内容属性，运营人员再人工补齐 SKC 图片、包装、价格折扣等信息，最后到 SHEIN 后台或开放平台发布。

已有资料确认：

- MDM 的 `SPU + SKU` 可以组合出商品信息导出维度的大部分字段，核心结构是 `SPU -> SKC -> SKU`。
- 深绘同步结果是商品内容包，包含标题、类目、颜色别名、图片、详情页、尺码表、属性和 SKU 相关内容。
- SHEIN 全托管发品需要先查询发布规范、属性模板和图片要求，再转换图片，最后调用发布接口并跟踪审核。
- SKC 图片在当前阶段允许人工上传补齐。
- 价格公式需要支持批量调整和折扣维护。
- 观远 BI 毛重、尺码转换表、包装规则、低倍率清单等第一期接受 Excel 上传和人工维护。
- 图片第一期以人工上传 SKC 维度图片包补齐为主，深绘主图和详情页作为补充来源。

因此第一版产品目标不是替代完整 ERP，而是把 SHEIN 全托管上新的关键链路产品化、数据化、可追踪。

## 3. 产品目标

### 3.1 MVP 目标

1. 将 MDM 和深绘数据自动汇集成 SKU 粒度的 SHEIN 上新草稿。
2. 支持人工补齐 SKC 图片、包装尺寸、毛重、类目映射和属性映射。
3. 支持供货折扣规则维护、批量价格试算和发布价格快照。
4. 支持发布前校验，明确缺失字段、图片、价格、尺码表、属性映射问题。
5. 支持调用 SHEIN 全托管发布接口，保存 SHEIN 返回的 `spu_name`、`skc_name`、`sku_code`。
6. 支持发布任务状态跟踪，回写审核通过或失败原因。

### 3.2 非目标

MVP 不覆盖以下能力：

- 多平台同时发布。
- TEMU 实际发布。
- SHEIN 订单、履约、库存、财务对账。
- 自动生成商品图片。
- 自动生成全部平台属性映射。
- 自动处理合规证书、警告语和标签打印。

## 4. 用户角色

| 角色 | 主要诉求 | 核心操作 |
| --- | --- | --- |
| 商品运营 | 批量准备上新资料，减少 Excel 人工整理 | 查看草稿、补图、选择类目、修正字段 |
| 价格运营 | 批量维护供货折扣和价格 | 配置折扣规则、试算、确认价格版本 |
| 平台运营 | 负责 SHEIN 发布和审核跟进 | 发布、查看失败原因、重新提交 |
| 系统管理员 | 维护平台账号和接口同步 | 配置 SHEIN 账号、查看任务日志 |

## 5. 核心概念

| 概念 | 定义 |
| --- | --- |
| SPU | 款级商品，来自 MDM，主键为 `款号` |
| SKC | 款色级商品，来自 SKU 派生或 MDM 字段，主键为 `SKC编码` |
| SKU | 款色码级商品，来自 MDM，主键为 MDM 真实 SKU 字段 |
| 企业码 | 内部码，不直接等同于 SHEIN 商家 SKU |
| 内容包 | 深绘同步出的标题、图片、详情页、尺码表、属性内容 |
| 刊登草稿 | 面向某个平台账号生成的待发布商品资料 |
| 发布链接 | SHEIN 侧一次可发布/可编辑的商品链接；一个款号默认一个发布链接，但同一 SKC 允许拆到多个发布链接 |
| 平台身份 | 平台发布成功后返回的对象 ID，例如 SHEIN 的 `spu_name/skc_name/sku_code` |
| 价格快照 | 某次发布时固化下来的价格计算结果 |

## 6. MVP 功能范围

### 6.1 数据同步

MDM 同步：

- 同步 SPU 接口数据到款级主数据。
- 同步 SKU 接口数据到 SKU 主数据。
- 从 SKU 派生 SKC 层级。
- 保留原始接口 payload、同步批次和来源时间。

深绘同步：

- 按款号或深绘商品 code 同步内容包。
- 抽取中文标题、英文标题、颜色、图片、详情页、尺码表、商品属性。
- 保留深绘原始 payload，便于字段追溯。
- 深绘和 MDM 的 SPU、SKC、SKU 字段本质对齐，允许在同步时做字段清洗和归一化。

Excel 规则导入：

- 观远 BI 毛重表支持 Excel 上传更新。
- SHEIN 尺码转换表支持 Excel 上传更新。
- 包装规则支持 Excel 上传、增删改查和批量应用。
- 低倍率款号清单支持 Excel 上传和版本化维护。
- SHEIN 末级类目映射不是 MDM 小类一对一关系，按 MDM 中类、小类、性别、年龄段综合匹配；规则支持 Excel 上传、界面批量更新和人工覆盖。

### 6.2 刊登草稿生成

系统按批次选择款号后生成 SHEIN 全托管上新草稿：

- 一个款号默认生成一个 `listing`。
- 如果业务需要同一个 SKC 拆成两个 SHEIN 链接发布，同一个款号可以生成多个 `listing` 发布单元。
- 一个 SKC 生成一条 `listing_skc`。
- 一个 SKU 生成一条 `listing_sku`。
- 标题、图片、尺码表优先取深绘。
- 主数据、颜色、尺码、挂牌价优先取 MDM。
- SKC 图片允许为空，但状态标记为待补齐。

### 6.3 人工补齐

支持运营人员在草稿页补齐：

- SKC 主图。
- SPU/SKC/SKU 图片类型归类。
- 包装尺寸和毛重。
- SHEIN 末级类目。
- SHEIN 属性映射。
- 商品标题、英文标题和描述修正。
- SHEIN 必填属性的枚举值或可填写范围，必须先从 SHEIN 发布规范和属性模板获取。
- 深绘和 MDM 能取到的数据优先自动填充，其余必填项支持批量填充和批量编辑。
- 非必填属性第一期允许为空。

### 6.4 价格规则

MVP 支持供货价规则：

```text
供货价 = 挂牌单价 * 供货折扣
建议零售价 = ROUND(挂牌单价 / 汇率, 精度)
```

折扣规则支持：

- 按年份，例如 `2026 = 0.4`。
- 按低倍率款号清单，例如命中则 `0.45`。
- 按品牌、季节、产品线、小类覆盖。
- 按批次手工覆盖。

规则优先级：

```text
手工 SKU 覆盖 > 款号清单 > 品牌/品类/季节规则 > 年份默认规则 > 系统默认值
```

发布时必须固化价格快照，后续规则变化不影响已发布任务。

价格字段需要人工确认后才允许进入发布流程。

### 6.5 发布前校验

校验分为阻断和警告。

阻断项：

- SKU 缺少 MDM 真实 SKU 字段或供应商 SKU。
- SKC 缺少主图。
- 缺少 SHEIN 末级类目。
- 缺少 SHEIN 必填属性。
- SHEIN 必填属性未命中可用枚举值或可填写范围。
- 缺少标题默认语种。
- 缺少供货价或币种。
- 缺少包装重量或尺寸。
- 同一店铺内供应商 SKU 重复。
- 图片未转换为 SHEIN 可用 URL。

警告项：

- 深绘标题为空但 MDM 名称可兜底。
- 某些非必填属性未填写。
- 图片尺寸可能不满足最佳规范。
- 毛重或尺寸使用默认规则。

### 6.6 SHEIN 发布

发布流程：

1. 查询 SHEIN 商品发布字段规范。
2. 查询 SHEIN 类目属性模板。
3. 校验草稿字段。
4. 上传或转换图片为 SHEIN URL。
5. 构造 `publishOrEdit` 请求。
6. 调用 SHEIN 发布接口。
7. 保存返回的 `spu_name`、`skc_name`、`sku_code`、`version`。
8. 通过轮询或 Webhook 跟踪审核。

发布和编辑都按版本管理。每次提交都生成独立发布版本、字段快照、价格快照、图片快照、请求 payload 和响应 payload。发布失败或审核失败后，修正草稿并生成新版本重新提交。

### 6.7 发布结果跟踪

发布结果需要支持：

- 任务级状态。
- SPU/SKC/SKU 级平台身份映射。
- 失败原因列表。
- 原始请求和响应留档。
- 支持失败后修正草稿并重新发布。

## 7. 页面与流程

### 7.1 上新批次列表

功能：

- 查看批次名称、款数、SKC 数、SKU 数。
- 查看待补齐、校验失败、待发布、发布中、审核中、已通过、已驳回数量。
- 支持创建批次、导入款号、从 MDM 查询款号。

### 7.2 刊登草稿工作台

功能：

- 按款号、SKC、SKU 查看草稿。
- 展示字段来源：MDM、深绘、规则、人工。
- 展示缺失项和校验错误。
- 支持批量补图、批量设置包装、批量应用价格规则。
- 支持预览即将提交给 SHEIN 的核心字段。
- 支持按批次批量应用类目映射、必填属性、包装规则、尺码转换和价格规则。

### 7.3 图片补齐页

功能：

- 按 SKC 上传主图。
- 标记图片用途：主图、细节图、方形图、色块图、详情图。
- 显示 SHEIN 图片规范检查结果。
- 显示图片是否已转换为 SHEIN URL。

### 7.4 价格规则页

功能：

- 维护折扣规则。
- 上传低倍率款号清单。
- 批量试算供货价和建议零售价。
- 保存价格版本。
- 将价格版本应用到上新批次。

### 7.5 发布任务页

功能：

- 展示任务状态机。
- 展示 SHEIN 请求、响应、traceId、错误码。
- 支持失败重试。
- 支持审核失败后回到草稿修正。

## 9. 未来多平台预留

TEMU 未来接入时，不改变核心商品模型，只新增平台适配器和平台字段映射。

SHEIN 与 TEMU 的主要共性：

- 都是 `SPU -> SKC -> SKU`。
- 都需要平台类目。
- 都需要类目属性模板。
- 都需要图片上传或转换。
- 都有平台返回的商品 ID。
- 都有价格、库存、审核或生命周期状态。

主要差异：

| 维度 | SHEIN | TEMU |
| --- | --- | --- |
| 接口形态 | REST path | `/openapi/router` + `type` |
| 发布接口 | `publishOrEdit` | `bg.glo.goods.add` |
| 平台 ID | `spu_name/skc_name/sku_code` | `productId/productSkcId/productSkuId` |
| 类目属性 | `category_id/product_type_id/attribute_id` | `cat1Id...cat9Id/pid/templatePid/specId` |
| 图片 | 上传或外链转换为 SHEIN URL | base64 上传为 TEMU URL |
| 尺码表 | 发布接口 `size_attribute_list` | 独立创建尺码表后引用 `businessId` |
| 全托价格 | 发布供货价，后续 `update-cost` | 申报价、核价、调价单流程 |

## 10. 验收标准

MVP 可验收条件：

1. 能调用 SHEIN 接口获取当前店铺可发布类目、发布字段规范和属性模板。
2. 能维护 MDM 中类、小类、性别、年龄段到 SHEIN 末级类目的组合映射规则，并批量应用到草稿。
3. 能从 MDM 和深绘生成 SHEIN 全托管上新草稿。
4. 能导入观远 BI 毛重、尺码转换表、包装规则和低倍率款号清单。
5. 能人工上传并绑定 SKC 图片包，并使用深绘主图和详情页作为补充素材。
6. 能维护折扣规则并批量生成价格快照，且价格字段经过人工确认。
7. 能对一个批次执行发布前校验，并输出字段级错误。
8. 测试阶段能构造 SHEIN 发布 payload，并调通 SHEIN 测试环境。
9. 能保存 SHEIN 返回的 `spu_name/skc_name/sku_code`。
10. 能展示发布任务状态、发布版本、请求响应快照和审核结果。
11. 能失败后修正草稿，生成新版本并重新发布。

## 11. 待确认事项

1. 手动上传图片包的目录结构、命名规则、图片用途识别方式和 SKC 匹配规则。
2. 同一 SKC 拆成多个 SHEIN 链接发布时的业务触发条件、拆分维度和界面操作方式。
3. SHEIN 全托管账号的默认语种、币种和类目权限是否固定。
4. SHEIN Webhook 是否能在测试环境开通；若不能，MVP 使用轮询。
5. SHEIN 接口签名、token 刷新、限流和错误码处理细节，以 SHEIN 开放平台实际文档和测试账号返回为准。
